## الخطوات التالية

### للنشر:
1. اضغط زر **Publish** في لوحة التحكم
2. انتظر اكتمال البناء (5-10 دقائق)
3. حمّل ملف APK
4. ثبّت على هاتفك

### للتطوير:
1. `npm install` - تثبيت المكتبات
2. `npm run android` - تشغيل على محاكي
3. `npm run build` - بناء APK
